# Frontend Application

React application built with Vite, React Router, and modern React patterns.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- React 18 with hooks
- React Router for navigation
- Context API for state management
- Axios for API calls
- Toast notifications
- Responsive design

## Project Structure

```
src/
├── components/    # Reusable components
├── context/       # React Context providers
├── pages/         # Page components
├── utils/         # Utility functions
└── tests/         # Test files
```

## Testing

```bash
npm test
```
